請直接到 **水球軟體學院課程平台**（[](https://world.waterballsa.tw/)[https://world.waterballsa.tw](https://world.waterballsa.tw)）研究我們目前課程平台的產品功能，然後**盡可能重製多少是多少**，開發的功能越多越好。

產品功能附錄/摘要在本文最底部。

## 什麼是「重製」？

所謂的重製，就是「重寫一個，功能越像越好」，完整意思是：

1. 瀏覽[水球軟體學院課程平台](https://world.waterballsa.tw)，了解這個平台提供了哪些功能。
2. 你可以用任何建模方法（Use Case Diagram、User Story Map、功能清單、心智圖等）來理解和整理這些功能。用你自己好思考的方式切入功能要點。
3. 自己決定開發範疇：在一週半的時間限制內，你可以自己決定要實作哪些功能、想要如何在面試中展現哪些能力。我們不會規定你一定要做哪些功能。
4. 開始開發：根據你的規劃，開始實作你的重製版本。
5. 有問題，就到我們公司的 [Discord 的 #需求訪談](https://discord.gg/Zk2hS5V7tg) 頻道中，與我們的產品經理進行角色扮演、提問來釐清需求。

## 我們的驗收標準

對我們來說，重製**的完整度越高、功能運作邏輯越相似，就代表你的能力越強**。

這代表：

- 你能主動釐清產品的運作原理（Discovery）
- 你能把產品功能規格化（Formulation）
- 你的規格驅動開發能力好（Automation）

你能透過 AI 放大你的產值，做到不可思議的開發速度

換句話說，重製完整度 = Discovery × Formulation × Automation 的綜合體現。

## 時間限制

你有 **一週半（10.5 天）** 的時間完成這個挑戰，從今天開始計算（請以面試官回信中的期限為主）。

我們不期待你在一週半內把整個平台都復刻完，那並不容易。我們要看的是：

- 在有限時間內，你如何取捨？
- 你選擇先做哪些功能？為什麼？
- 你的開發速度和品質如何？
- 你是否主動溝通和提問？
- 你是否展現出規格思維？
- 你是否善用 AI 來達到多倍產能？

## 主導敏捷開發、多次 Release 的能力

1. 我們不期望你全部開發完，如果你真的能全部開發完的話，你是神。
2. 因此，我們要的是你的敏捷開發以及規劃的能力。決策和取捨在你身上，你要做自己的 TPM，去切多次 Release。第一次 Release 範疇是什麼？第二次？第三次？核心到延伸功能？快速得到 Feedback？快速得到規格的驗證？
3. **務必以此為目標 👉🏻：**快速確認需求 > 快速澄清規格（最好是可執行規格，如果你不知道什麼是可執行規格，那你必須去學） > 快速收斂 release 1 範疇 > 快速 AI 開發 > 快速上線聽取用戶回饋 > 從回饋中快速修正規格進行第二次迭代 > 快速收斂 release 2 範疇 > …
4. 能在一週內做到「多次 Release、範疇清楚、結果精準」 的人，會是最厲害的 AI 敏捷開發高手。

### Release 是什麼？

- 你可以直接把「錄影」丟到「需求訪談」頻道中，來與 PM/PO 針對某部分範疇互動。
- 也可以直接將需求上線，給 PM/PO 連結來讓對方可以測試。

<aside>

記得，敏捷開發的精神是快速迭代，而不是埋頭苦幹。

</aside>

## 技術要求

1. 前端用 Next.js
2. 後端用 Java / Spring Boot，資料庫用 Postgresql
3. 前後端以及資料庫全部但要用 Docker-Compose 包起來，讓我們能夠可以用 docker-compose 指令在本地執行。
4. 前後端中涵蓋的其他技術、套件不限制。
5. 如有使用到：雲端服務，則雲端服務不限，我們的課程平台使用 S3 來儲存影片資料，您可以尋求其他手段來部署影音資料。

### 我們期待看到

- 前後端都有實作（至少基本的全端開發）
- 有一定的 Code 品質（至少有基本分層就好，請勿做 clean-arch 等進階架構設計，你絕對沒有時間做這個，然後面試官也絕對比你熟這個，是危險地帶）
- 一定要有 E2E 自動化測試，單元測試隨意
- 如果有寫規格文件會加分（e.g., specs, gherkins, …)

## 關於作品擁有權

**你開發的作品的擁有權在你自己身上**，寫完就可以放進自己的作品集中。

是否要繼續面試的決定權也在你手上：）

# 第二關：線上面試

## 面試邀請

1. 當求職者透過預約連結，完成預約後，兩位技術面試官會自動收到面試邀請。
2. 確認面試時間無衝突，若有衝突需立即與求職者協調改期。

## 第二關面試內容

1. 第二關面試中，我們會問幾個知名的人格特質以及工程能力的問題。
2. 接著大多數時間就會去請你 Demo 你在第一關中所寫的「程式」（不是產品，產品已經在第一關請你桌錄 Demo 過了），此時我們想看的是你的程式的闡述能力。你必須了解 AI 所開發的程式，並且能有條有理地闡述它。
3. 最後會問你團隊憧憬和遠端協作上的一些問題。
4. 上述完成後，水球就會在 10 分鐘之內給您一個「薪水報價」，當下就會給您回饋並且談完薪水和期望報酬的部分。

<aside>

不必太緊張，只要你比你的競爭對手還強就好。

我們刻意將面試難度提高，只是為了讓有鑑別度的人能有所發揮。

</aside>

# 第二關結束後 — 等待結果

1. 第二關面試結束後，接著要等待 1~2 週，我們會在 1~2 周內公告面試結果。由於面試者人數可能多可能少，我們需要謹慎評估人選，因此會花一些時間，請見諒。
